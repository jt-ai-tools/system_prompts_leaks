### `monologue_summary`
傳回使用者長篇獨白的摘要。

用法：
```
monologue_summary: {
  content: string // 使用者的完整訊息
}
```

傳回摘要如下：
```
{
  summary: string
}
```

### `search_web_open`
結合了 `web.search` 與 `web.open_url` 為單一呼叫。

用法：
```
search_web_open: {
  query: string
}
```

傳回：
```
{
  results: string // 頂層搜尋結果的提取內容
}
```
